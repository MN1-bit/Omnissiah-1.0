## 생성된 디렉토리
```
logs/
├── system/     시스템 로그 (.log)
├── trades/     거래 로그 (.json)
└── errors/     에러 로그 (.log)
```

## 구현된 기능

### OmnissiahLogger 클래스
| 메서드 | 기능 |
|--------|------|
| `info()` | 정보 로그 |
| `debug()` | 디버그 로그 |
| `warning()` | 경고 로그 |
| `error()` | 에러 로그 (에러 파일에도 저장) |
| `log_trade()` | 거래 로그 (JSON) |
| `cleanup_old_logs()` | 30일 이상 로그 삭제 |

### 로그 포맷

```
[2024-12-16 10:30:05] [WARNING] 경고 메시지
```
