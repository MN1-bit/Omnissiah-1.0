## 생성된 파일
- `strategy/green_mode.py` - 평균 회귀 전략
- `strategy/red_mode.py` - 추세 추종 전략
- `strategy/black_mode.py` - 방어 전략

## 구현된 전략

### Green Mode (평균 회귀)
| 항목 | 내용 |
|------|------|
| 조건 | Z-Score < 1.0 |
| 진입 | 가격 ≤ Lower Band |
| 청산 | 가격 ≥ VWAP |
| 특수 | 15:50 전량 청산 |

### Red Mode (추세 추종)
| 항목 | 내용 |
|------|------|
| 조건 | 골디락스 (KER>0.3, ADX>25) |
| 진입 | 전일 고가 돌파 |
| 피라미딩 | 최대 3회 (+1%) |
| 청산 | MA20 이탈 |

### Black Mode (방어)
| 항목 | 내용 |
|------|------|
| 조건 | Z-Score ≥ 2.0 또는 백워데이션 |
| 기본 | 전량 청산 |
| 인버스 | 백워데이션 + 2PM + 신저점 시 |
| 보유 | 인버스 최대 3일 |

## 테스트 결과
```
✅ Green Mode: VWAP 밴드, BUY/SELL 신호 정상
✅ Red Mode: 돌파 진입, 피라미딩, MA 청산 정상
✅ Black Mode: 전량 청산, 인버스 진입/청산 정상
```
