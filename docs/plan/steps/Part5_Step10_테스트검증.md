# Part 5 - Step 10: 테스트 및 검증

## 상태: 🟡 진행 중

## 목표
전체 시스템 통합 테스트 및 검증

## 테스트 항목

### 1. GUI 응답성
- [ ] time.sleep() 사용 없음 확인
- [ ] 1분간 GUI 멈춤 없이 동작
- [ ] 로그 실시간 출력

### 2. IBKR 연결
- [ ] IBKR Gateway 꺼진 상태 → "연결 실패" 메시지
- [ ] IBKR Gateway 켠 후 시작 → "연결됨" 표시
- [ ] 연결/해제 시 GUI 상태 반영

### 3. 킬 스위치
- [ ] 매 루프마다 가장 먼저 체크
- [ ] 발동 시 명확한 GUI 표시

### 4. 주문 승인
- [ ] 모든 주문이 approve_order() 통과
- [ ] 거부 시 로그에 명확히 기록

### 5. 레짐 판단
- [ ] 횡보/상승/위기 정상 표시
- [ ] 색상 (Yellow/Cyan/Red) 정확

### 6. 하이브리드 업데이트
- [ ] 5초 기본 주기 동작
- [ ] |Z-Score| >= 1.0 시 1초 전환
- [ ] 일봉 캐시 1일 1회 갱신

## 테스트 시나리오
1. IBKR Gateway 없이 실행 → 연결 실패 로그
2. Start 버튼 → 하이브리드 루프 시작
3. 1분간 동작 확인
4. 각 모드 색상 확인
5. Stop 버튼 → 정상 종료

## 완료 조건
- 모든 체크리스트 통과
- 버그 없이 1분간 안정 동작
