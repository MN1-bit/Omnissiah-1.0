# Part 6 - Step 12: 자동 시작/종료 스케줄러

## 상태: 🟡 진행 중

## 목표
장 시작/종료 시간에 따라 자동으로 시스템을 관리하는 스케줄러 구현

## 핵심 기능

### 1. 시장 시간 관리
| 기능 | 시간 (미국 동부) |
|------|-----------------|
| 프리마켓 | 04:00 ~ 09:30 |
| 정규장 | 09:30 ~ 16:00 |
| 애프터마켓 | 16:00 ~ 20:00 |

### 2. 자동 동작
| 이벤트 | 동작 |
|--------|------|
| 장 시작 15분 전 | 시스템 준비, 데이터 갱신 |
| 장 시작 | 전략 활성화 |
| 15:50 | 청산 여부 결정 |
| 장 마감 | 전략 비활성화 |

### 3. 레짐별 청산 규칙
| 레짐 | 15:50 청산 규칙 |
|------|-----------------|
| 횡보 | 전량 청산 |
| 상승 | 트레일링 스탑 유지 |
| 위기 | 즉시 청산 |

### 4. 휴장일 처리
- pandas_market_calendars 사용
- 휴장일 자동 감지 및 스킵
