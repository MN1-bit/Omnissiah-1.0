# Part 7 - Step 16.5: 대시보드 통합

## 상태: 🟡 진행 예정

## 목표
차트 위젯, 거래 내역 패널, 로깅 시스템을 메인 대시보드에 통합

## 통합 대상

### 1. 차트 위젯 (`gui/chart_widget.py`)
- LiveChartWidget 클래스
- 실시간 가격 차트
- VWAP 밴드
- 포지션 마커

### 2. 거래 내역 패널 (`gui/trade_panel.py`)
- TradeHistoryPanel 클래스
- 거래 테이블
- 일일 요약

### 3. 로깅 시스템 (`core/logger.py`)
- OmnissiahLogger 싱글톤
- GUI 로그 → 파일 동시 저장
